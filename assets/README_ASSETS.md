# Etherea v0.2.2 Asset Patch

Small 32x32 transparent PNG assets for **Etherea: Ashes of the Saints — Pocket Roguelike**.

## Included

### Tavern NPCs
- Tavern Keeper
- Omar Hafez, Creator of Etherea
- Verdan Thorne, Keeper of Beasts
- Azael Vire, Blade for Hire

### Early enemies
- Zombie
- Skeleton
- Goblin

### Tavern / hub tiles
- Bed / rest point
- Expedition board
- Lore bookshelf
- Hollow Quill notice / inheritance board
- Hearth / fireplace
- Tavern floor

### Area / dungeon tiles
- Foundries furnace
- Deeper Well marker
- Exit rune

## Notes

These are simple pixel-art-style placeholders intended to match the current dark fantasy Tkinter look without requiring pygame or external libraries.

Recommended destination:

`assets/sprites/`

Codex can wire these into the current asset loader by mapping the filenames to the appropriate tile/enemy/NPC identifiers.
